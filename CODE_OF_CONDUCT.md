# Code of Conduct

MILF should be a welcoming, respectful place to collaborate.

## Expected Behavior

- Be kind, patient, and constructive.
- Assume good intent while still naming problems clearly.
- Respect different experience levels, backgrounds, and communication styles.
- Keep feedback focused on the work, not the person.
- Help make issues and pull requests understandable for future contributors.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Sexualized, discriminatory, or exclusionary language or behavior.
- Trolling, sustained disruption, or bad-faith participation.
- Publishing private information without explicit permission.
- Repeatedly ignoring maintainer boundaries or moderation requests.

## Enforcement

Maintainers may remove comments, close issues, block participants, or take other reasonable action to protect the project community.

To report a conduct concern, contact the maintainer privately using the contact method listed in `SECURITY.md` or the maintainer's GitHub profile. Reports will be handled with care and discretion.

## Scope

This code of conduct applies in project spaces, including GitHub issues, pull requests, discussions, releases, and any other community spaces connected to MILF.
